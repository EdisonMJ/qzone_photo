批量下载QQ空间相册。

目前支持下载的包括：
1. 对所有人公开的相册；
2. 好友中好友可见的相册。

# 使用方法
* 安装[Python 2.7.13](https://www.python.org/downloads/release/python-2713/)；
* 安装依赖：pip install -r requirements.txt；
* 修改main.py的登录账户为你的qq账号和密码；
* 修改main.py的qq号为你需要的qq号；
* cmd中键入Python main.py运行。
